# aptos-ml-project
High-performance deep learning pipeline for APTOS 2019 diabetic retinopathy detection.  Uses EfficientNetB3 backbone, stratified K-fold cross-validation, custom ordinal loss, advanced image preprocessing, and test-time augmentation for improved accuracy and robustness.

# APTOS Blindness Detection

Deep learning model using EfficientNetB3 for diabetic retinopathy classification.

## Features
- Stratified K-Fold training
- Custom ordinal loss
- Advanced preprocessing
- Test-time augmentation
